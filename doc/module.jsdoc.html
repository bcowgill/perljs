/**
	Demonstration of simple jsdoc module definition. If the name of the module is omitted, it is derived from the filename.

/** simplest documentation of an internal module member */

/** indicate a private module member so it won't be documented
	@private
*/

	@member {number} value - initial value counter

var value = 1;
